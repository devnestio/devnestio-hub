About UUID Versions

v4 — Random
128 bits of cryptographically random data. Version bits are 4 and variant bits are set. Most commonly used. Zero collision risk in practice.
v1 — Time-based
Combines the current timestamp (100-nanosecond intervals since Oct 1582), a random clock sequence, and a node identifier. Sortable by time but leaks the MAC/timestamp.
v5 — Name-based (SHA-1)
Deterministic: given the same namespace + name, always produces the same UUID. Uses SHA-1 internally. Useful for stable IDs from known strings (e.g. URLs, DNS names).
